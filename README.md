# SecSnail
Secure data through snailmail

## Build
Run:
```
yarn
yarn global add browserify
browserify ./src/main.js > ./public/bundle.js 
```
or
```
npm
npm install -g browserify
browserify ./src/main.js > ./public/bundle.js 
```
